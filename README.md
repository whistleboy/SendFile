# SendFile
This is a file transfering Client and Server.

### 语言
文本介绍：中文
开发：纯英文

### 项目介绍
本项目主要用于内网中文件的传输。

### 项目特点
- 一对多的传输
- 断点传输
- 多文件同时传输
- 发送端和接收端都在同一个ui界面，易于操作

- Windows：解压后，直接执行exe文件即可
- Linux：未实现

### 使用教程
#### 接收端：
- 点击Reciever的tab页
- 点击右下角的Add File Path，用于指定接受文件的存储目录
- 确定后等待接收

#### 发送端：
- 点击Sender的tab页
- 点击Choose按钮，在输入框中输入传输目标的IP，确定
- 点击Choose FIle[s]，选择要传输的文件，确定
- 点击Send
